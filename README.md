# Proyecto

Este proyecto usa una memoria portable basada en archivos Markdown dentro de `/cerebro`.

## Estructura importante

- `/cerebro/decisiones.md`: decisiones importantes
- `/cerebro/people.md`: personas, roles y contactos
- `/cerebro/preferences.md`: preferencias y convenciones
- `/cerebro/user.md`: contexto y objetivos del usuario
- `/cerebro/personality.md`: personalidad operativa del compañero
- `/cerebro/changelog.md`: bitácora de cambios significativos
- `/active.md`: trabajo en curso, tareas, pruebas y errores

## Regla general

Antes de actuar, revisar `/cerebro` y `/active.md`.
